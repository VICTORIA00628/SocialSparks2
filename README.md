# SocialSparks

## About
SocialSparks is an Android app that suggests social interactions based on the time of day.

## Features
- Enter time of day (Morning, Afternoon, Dinner, Night)
- Get social spark suggestions
- Reset button to clear input

## How to use
1. Type the time of day
2. Click "Get My Spark"
3. Read your suggestion!

## Built with
- Kotlin
- Android Studio
- GitHub Actions
